# Instructions pour les utilisateurs bloqués

## Si tu ne peux pas te connecter :

### Option 1 : Fix automatique (recommandé)
1. Va sur : https://tonsite.com/auth-fix
2. Attends que la page te redirige automatiquement
3. Reconnecte-toi normalement

### Option 2 : Manuel
1. Ouvre les outils de développement (F12)
2. Va dans "Application" > "Cookies"
3. Supprime tous les cookies
4. Refresh la page
5. Reconnecte-toi

### Option 3 : Navigation privée
1. Ouvre une fenêtre de navigation privée
2. Connecte-toi normalement
3. Ça devrait fonctionner

## Pourquoi ce problème ?

Nous avons effectué une mise à jour importante qui a corrompu les anciennes sessions.
Désolé pour le désagrément, c'est un problème temporaire qui se résout en 30 secondes !
